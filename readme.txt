Wather Station based on ESP8266.
In folder ESP8266 is code for this devices.
In folder WindAngleV1 is code for arduino to make new sensor based on digital compas, magnetic encoder and accelerometer.
In folder WindSpeed is code for arduino to make new sensor based on hall sensor
